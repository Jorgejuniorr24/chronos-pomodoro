## 📌 Sobre o Projeto

O **Chronos Pomodoro** é uma aplicação web de produtividade baseada na técnica Pomodoro — um método de gerenciamento de tempo que divide o trabalho em blocos de **25 minutos focados**, seguidos de pausas estratégicas.

Desenvolvida com **React**, **TypeScript** e **Vite**, com foco em usabilidade e boas práticas de desenvolvimento front-end.

---

## 🚀 Tecnologias Utilizadas

* ⚛️ React
* 🔷 TypeScript
* ⚡ Vite
* 🎨 CSS3
* 📦 ESLint

---


## 🧠 Como Funciona a Técnica Pomodoro

1. Trabalhe por **25 minutos** com foco total
2. Faça uma **pausa curta** de 5 minutos
3. A cada **4 ciclos**, faça uma pausa maior de 15–30 minutos
4. Repita e aumente sua produtividade 🚀

## 💻 Como Rodar o Projeto

### 📥 Clonando o repositório
```bash
git clone https://github.com/seu-usuario/chronos-pomodoro.git
```

### 📦 Instalando dependências
```bash
cd chronos-pomodoro
npm install
```

### ▶️ Rodando o projeto
```bash
npm run dev
```

A aplicação estará disponível em:
👉 http://localhost:5173
